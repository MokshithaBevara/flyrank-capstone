# FlyRank Capstone

> A learning project for the **FlyRank AI Internship** — Frontend AI Engineering track.

## Project Overview

This repository is my capstone project for the FlyRank AI Internship. The goal is to build frontend skills while working with AI-assisted development tools.

**What this project is about:**
- Learning how to design and build web interfaces
- Practicing clean, readable code with help from AI tools
- Growing the project step by step throughout the internship

**Current status:** The repository is in its early stages. The tech stack and folder structure will expand as the project develops.

## Setup Instructions

Follow these steps to get the project on your machine.

### Prerequisites

- [Git](https://git-scm.com/downloads) installed on your computer
- A code editor such as [VS Code](https://code.visualstudio.com/) or [Cursor](https://cursor.com/)

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/mokshithabevara/flyrank-capstone.git
   cd flyrank-capstone
   ```

2. **Open the project in your editor**
   - In VS Code or Cursor: **File → Open Folder** → select the `flyrank-capstone` folder

3. **Start building**
   - There is no install step yet — the project will add one when it uses npm or another package manager.
